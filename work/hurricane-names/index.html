Hurricane names

Not every name can become a hurricane. The World Meteorological Organization favors names that are short, easy to pronounce, distinctive and free of controversy so they’re clear in emergencies. Type a name to see how it stacks up against the same criteria used to build official storm lists.

Live project
Readers can test whether a name fits hurricane-naming rules by typing it into a search box.
The tool evaluates each name against key World Meteorological Organization criteria — length, pronunciation, uniqueness and non-notability — and explains where it passes or fails.

A winding timeline traces retired Atlantic storm names from the 1950s through the 1970s, when hurricanes were exclusively given women’s names.
